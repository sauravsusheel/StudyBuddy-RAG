## FAQ for GitHub

### Q: How is this different from ChatGPT?
A: StudyBuddy provides **exact source citations** — you'll know exactly which 
page and text the answer came from. It also keeps your data private (no cloud 
storage) and lets you choose from 10+ AI models.

### Q: Do I need to pay?
A: StudyBuddy itself is free and open source. You only pay for API calls to 
OpenRouter (~$5-10/month for typical usage).

### Q: Will my documents be uploaded to the cloud?
A: No. All processing happens locally on your machine. Only your query goes 
to OpenRouter's API (encrypted).

### Q: Can I self-host this?
A: Yes! The project is fully open source. You can run it locally, deploy to 
Docker, or use any hosting platform.

### Q: What languages does it support?
A: Python backend, vanilla JavaScript frontend. Any language pair supported 
by the LLM models (100+ languages via OpenRouter).

### Q: How large can documents be?
A: Tested up to 500-page PDFs. Chunking happens locally, so size depends 
mainly on your RAM.
